# Handwritten Digit Recognizer (MNIST)

A minimal, modern web app that recognizes handwritten digits (0–9) using a
neural network trained on the classic MNIST dataset.

- **Backend:** Python + Flask
- **ML:** TensorFlow / Keras (Sequential model, ~98% test accuracy)
- **Frontend:** HTML + CSS + JavaScript (Canvas API)

## Features

- ✏️ **Draw a digit** on a 280×280 canvas (mouse or touch)
- 📤 **Upload an image** of a handwritten digit
- 🎯 **Predicted digit** + **confidence score**
- 📊 **Probability bars** for all 10 digits
- ⚡ **Real-time prediction** while drawing (toggleable)
- 🧹 **Clear Canvas** and **Predict** buttons
- 🎨 Clean, dark, minimal UI

## Running it in VS Code (step by step)

### 1. Open the project in VS Code

```
code digit_recognizer
```

### 2. Create and activate a virtual environment

**Windows (PowerShell):**
```powershell
python -m venv .venv
.venv\Scripts\Activate.ps1
```

**macOS / Linux:**
```bash
python3 -m venv .venv
source .venv/bin/activate
```

VS Code should prompt you to select the new interpreter (`.venv`). If not,
press `Ctrl+Shift+P` → **Python: Select Interpreter** → pick `.venv`.

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Train the model (only needed once)

```bash
python train_model.py
```

This downloads the MNIST dataset (~11 MB), trains for 10 epochs (~2–5 min on
CPU), and saves the model to `model/mnist_model.h5`. Expected test
accuracy: **~98%**.

### 5. Start the Flask app

```bash
python app.py
```

You should see something like:
```
 * Running on http://127.0.0.1:5000
```

### 6. Open the app

Open <http://127.0.0.1:5000> in your browser. Draw a digit, upload an
image, or toggle real-time prediction.

## Model architecture

```
Input  (28, 28)
Flatten
Dense  256, ReLU
Dropout 0.2
Dense  128, ReLU
Dropout 0.2
Dense   10, Softmax
```

- Optimizer: Adam
- Loss: Sparse categorical cross-entropy
- Epochs: 10, batch size 128

## API

`POST /predict` — JSON body or multipart form upload.

**Canvas (base64) request:**
```json
{ "image": "data:image/png;base64,iVBORw0KGgo..." }
```

**File upload:** send a `file` field via multipart form.

**Response:**
```json
{
  "digit": 7,
  "confidence": 0.9942,
  "probabilities": [0.0001, 0.0001, 0.0003, 0.0002, 0.0004, 0.0010, 0.0005, 0.9942, 0.0020, 0.0012]
}
```

## Tips

- **Black ink on white background** works best — that's how MNIST was
  preprocessed. The backend automatically inverts and centers your input.
- **Draw thick** — MNIST digits use bold strokes.
- For uploads, square cropped images of a single digit work best.

---

## Troubleshooting

- **`Trained model not found...`** — run `python train_model.py` first.
- **TensorFlow install fails on Apple Silicon** — use
  `pip install tensorflow-macos` instead.
- **Port 5000 in use** — change the port in `app.py` (last line).
